# 🍽️ Sabor Express

Sistema de gerenciamento de restaurantes desenvolvido em Python para execução no terminal.

## 📋 Sobre o Projeto

O Sabor Express permite cadastrar restaurantes, listar os estabelecimentos cadastrados e alterar seu status entre ativo e inativo através de um menu interativo.

## 🚀 Funcionalidades

* Cadastro de novos restaurantes
* Listagem de restaurantes cadastrados
* Ativação e desativação de restaurantes
* Menu interativo no terminal
* Tratamento de opções inválidas

## 🛠️ Tecnologias Utilizadas

* Python 3
* Biblioteca `os`

## 📂 Estrutura dos Dados

Cada restaurante é armazenado com as seguintes informações:

* Nome
* Categoria
* Status (Ativo ou Desativado)

## ▶️ Como Executar

1. Certifique-se de ter o Python instalado.
2. Baixe ou clone o projeto.
3. Execute o arquivo principal:

```bash
python app.py
```

## 📌 Menu Principal

```text
1. Cadastrar restaurante
2. Listar restaurante
3. Alternar estado do restaurante
4. Sair
```

### 📚 Exercícios com Dicionários

O arquivo `exercicios_dicionarios.py` reúne uma série de exercícios práticos desenvolvidos para reforçar o uso de dicionários em Python. Durante os exercícios são trabalhados conceitos fundamentais da linguagem, como manipulação de dados, estruturas de repetição, condicionais e compreensão de dicionários.

Os exercícios abordam:

* Pesquisa de registros em uma lista de dicionários.
* Alteração, adição e remoção de chaves em um dicionário.
* Criação de dicionários utilizando Dictionary Comprehension.
* Verificação da existência de chaves.
* Contagem da frequência de palavras em uma frase utilizando dicionários.

O objetivo do arquivo é consolidar o entendimento sobre estruturas de dados do tipo dicionário, amplamente utilizadas para armazenar e organizar informações em Python.
